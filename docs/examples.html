Real-Time

Polling / Live Updates

<div p-request="GET /api/notifications
     > this" p-trigger="every 5s">
  No new notifications
</div>

Server:

app.get('/api/notifications', (req, res) => {
  const n = getUnread(req.user);
  res.send(n.length === 0
    ? '<p>No new notifications</p>'
    : n.map(x => `<div>${x.message}</div>`).join(''));
});

Todo List (CRUD)

<form p-request="POST /api/todos {this}
  > #todo-list.append :validate"
  p-on="pulse:afterSwap: this.reset()">
  <input name="title" required
         placeholder="New todo...">
  <button type="submit">Add</button>
</form>
<ul id="todo-list">
  <li id="todo-1">
    <span>Buy groceries</span>
    <button p-request="DELETE /api/todos/1
      > #todo-1.remove
      :confirm('Delete?')">X</button>
  </li>
</ul>

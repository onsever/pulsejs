Pulse.js

A lightweight, declarative library for building dynamic HTML applications. 8 attributes. One unified syntax.

What Is Pulse.js?

Pulse.js lets you build dynamic HTML applications with simple declarative attributes — no JavaScript required. It sends AJAX requests, swaps content into the DOM, and handles events, history, and more, all from your HTML.

The core innovation is the p-request attribute, which encodes method, URL, headers, body, target, swap behavior, and modifiers into one readable string.

<button p-request="DELETE /item/1 > closest tr.outer :confirm('Sure?')">
  Delete
</button>

Installation

CDN

<script src="https://unpkg.com/@onsever/pulse-js@1.0.0/dist/pulse.min.js"></script>

npm

npm install @onsever/pulse-js

ESM Import

import '@onsever/pulse-js';

Pulse auto-initializes on DOMContentLoaded. No setup code needed for basic usage.

Your First Example

<!-- Click the button, GET /hello, replace button contents -->
<button p-request="GET /hello">
  Click me
</button>

That's it. When clicked, Pulse sends a GET request to /hello and replaces the button's inner HTML with the response.
